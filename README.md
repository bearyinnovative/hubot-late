# hubot-late

record every late in your team

See [`src/late.coffee`](src/late.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-late --save`

Then add **hubot-late** to your `external-scripts.json`:

```json
[
  "hubot-late"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
